Camouflaged Instance Segmentation via Explicit De-Camouflaging

DCNet은 COD에서 한 단계 더 나아가 instance segmentation을 본격적으로 다룹니다. camouflage characteristic을 explicit하게 분리해 내는 de-camouflaging 관점이 핵심입니다.

2026.03.14

  • cod

오버뷰

DCNet은 COD에서 한 단계 더 나아가 instance segmentation을 본격적으로 다룹니다. camouflage characteristic을 explicit하게 분리해 내는 de-camouflaging 관점이 핵심입니다.

핵심 주장

  • camouflaged instance segmentation은 단순 binary COD보다 더 강한 camouflage suppression이 필요하다.
  • pixel-level camouflage decoupling과 instance-level camouflage suppression을 함께 써야 한다.
  • COD10K와 NC4K에서 average precision이 5% 이상 올라간다.

초록

이 논문은 Fourier transformation 기반 camouflage characteristic extraction과 difference attention으로 de-camouflaged feature를 만들고, reference point 기반 prototype similarity로 background noise를 억제하는 DCNet을 제안합니다.

서론

binary mask 수준에서는 객체만 찾으면 되지만, instance segmentation은 object-wise separation까지 해야 해서 camouflage가 더 까다롭게 작동합니다. 이 논문은 그 차이를 분명히 봅니다.

본론

저자들은 camouflage characteristic을 제거된 상태의 feature를 만들면 instance grouping까지 더 정확해질 수 있다고 주장합니다. 즉 먼저 de-camouflage하고, 그 다음 instance를 나누는 흐름입니다.

제안방법

pixel-level camouflage decoupling module이 camouflage characteristic을 빼고, instance-level suppression module이 reference similarity를 이용해 noisy prototype-pixel interaction을 줄입니다.

실험

대표 결과는 두 benchmark에서의 AP 향상입니다. binary COD가 아닌 instance-level evaluation에서 큰 폭의 개선을 보인다는 점이 중요합니다.

결론

DCNet은 camouflage를 먼저 벗겨낸 뒤 instance를 나누는 관점을 선명하게 보여준 작업입니다.

논의

CIS는 아직 논문 수가 많지 않아서, 이 논문은 instance-level camouflage 연구를 볼 때 거의 기준점처럼 읽힙니다.
