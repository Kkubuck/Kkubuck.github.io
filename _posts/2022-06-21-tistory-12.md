<h2>1. 스택이 비었는지 확인하는 함수</h2>
<pre class="lasso"><code>if self.top == self.size - 1:
            return True
        else:
            return False
</code></pre>
<h2>2. 스택이 꽉 차있는지 확인하는 함수</h2>
<pre class="python"><code>def is_stack_empty(self):
        if self.top == -1:
            return True
        else:
            return False
</code></pre>
<h2>3. push 함수</h2>
<pre class="ruby"><code>def push(self, data):
        if (self.is_stack_full()):
            return 
        self.top += 1
        self.stack[self.top] = data
</code></pre>
<h2>4. pop 함수</h2>
<pre class="ruby"><code>def pop(self):
        if (self.is_stack_empty()):
            return None
        data = self.stack[self.top]
        self.stack[self.top] = None
        self.top -= 1
        return data
</code></pre>
<h2>5. peek 함수</h2>
<pre class="ruby"><code>def peek(self):
        if self.is_stack_empty():
            return None
        return self.stack[self.top]
</code></pre>
<h2>6. 실습 push, pop 확인하는 함수</h2>
<pre class="swift"><code>def validate_stack_sequences(self, pushed, popped):
        i = 0
        c = 0
        while True: 
            if i &lt; len(pushed):
                self.push(pushed[i])
                if self.stack[self.top] == popped[c]:
                    self.pop()
                    c+=1
                i+=1
                
            if i &gt;=len(pushed):
                if self.stack[self.top] == popped[c]: 
                    self.pop()
                    c+=1
                else:
                    return False
                if (self.is_stack_empty()):
                        return True
</code></pre>
<h2>7. 스택 원리</h2>
<ul>
<li>스택은 <b>선입후출(FILO)</b> 구조이다.</li>
<li>통로가 <b>하나</b>밖에 없다.</li>
<li>스택에서 최상단의 값을 <b>top</b>이라고 한다.</li>
<li>스택에 데이터를 삽입하는 작동을 <b>push</b>라고 한다.</li>
<li>스택에서 데이터를 추출하는 작동을 <b>pop</b>라고 한다.</li>
<li>스택에서 <b>top</b>값을 확인하는 작동을 <b>peek</b>라고 한다.</li>
</ul>
