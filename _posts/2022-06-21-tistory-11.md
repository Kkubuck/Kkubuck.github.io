<h2>1. 노드 출력</h2>
<pre class="sql"><code>def printNodes(start):
    current = start
    if current == None:
        return
    print(current.data, end = ' ')
    while current.link != None:
        current = current.link
        print(current.data, end =' ')
</code></pre>

<h2>3. 노드 삭제 2단계 (첫 번째, 나머지)</h2>
<pre class="properties"><code>def deleteNode(deleteData):
    global memory, head, current, pre

    #첫 번째 노드 삭제
    if head.data ==deleteData:
        current = head
        head = head.link
        del current
        return
    
    #나머지 노드 삭제
    current = head
    while current.link != None:
        pre = current
        current = current.link
        if current.data == deleteData:
            pre.link = current.link
            del current
            return
</code></pre>
