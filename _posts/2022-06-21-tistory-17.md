<h2>1. 선택 정렬(앞부터 i, i+1 비교) O(n^2)</h2>
<pre class="sas"><code>def selectionSort(array):

    n = len(array)
    for i in range(n-1):
        minIdx = i
        for k in range(i + 1, n):
            if (array[minIdx] &gt; array[k]):
                minIdx = k
        array[i], array[minIdx] = array[minIdx], array[i]

    return array
</code></pre>
<p><b>i(0, n-1):</b> min값에 i를 저장, 한 사이클 끝나면 min과 i의 값을 스왑</p>
<p><b>k(i+1, n):</b> k값과 min값을 비교하면서 min값보다 k값이 작으면 min에 k값 대입</p>
<p><b>= i와 i+1값을 비교해 값을 바꿔주는 정렬</b></p>

<pre class="angelscript"><code>def insertionSort(array):
   
    n = len(array)
    for end in range(n):
        for cur in range(end, 0, -1):
            if(array[cur - 1] &gt; array[cur]):
                array[cur - 1], array[cur] = array[cur], array[cur - 1]
    return array
</code></pre>
<p><b>end(1, n):</b> cur이 비교할 범위를 정해준다. 처음 1 &rarr; <b>늘려가며 비교</b></p>
<p><b>cur(end, 0, -1):</b> end부터 0까지 -1하면서 내려오고 만약 cur-1값이 cur값보다 크면 cur-1과 cur을 스왑</p>
<p><b>= cur이 한번 움직일 때 마다 비교하고 cur-1값이 cur보다 크면 스왑</b></p>

<pre class="angelscript"><code>def bubble_sort(array):
   
    n = len(array)
    for i, end in enumerate(range(n - 1, 0, -1)):
        is_change = False
        for curr in range(0, end):
            if array[curr] &gt; array[curr + 1]:
                array[curr], array[curr + 1] = array[curr + 1], array[curr]
                is_change = True 
        print(i + 1, "번째 사이클: ", array)
        if not is_change:
            break
    return array
</code></pre>
<p><b>end(n-1, 0, -1):</b> curr이 비교할 범위를 정해준다. 처음 n-1 &rarr; <b>줄여가며 비교</b></p>
<p><b>curr(0, end):</b> 0부터 end까지 비교하면서 올라가고 만약 curr값이 curr-1값보다 크면 curr-1과 curr값을 스왑</p>
<p>= 스왑 할 시 플래그 값을 변경시켜 스왑했다고 알림, end사이클 1번 반복 할 동안 <b>한번도 스왑이 일어나지 않았다면</b> <b>정렬된 리스트</b>이기 때문에 반복문 탈출</p>
<p>= cur 한 사이클을 돌면 <b>제일 큰 값</b>이 <b>맨 뒤</b>에 놓인다. 계속 반복시 완전히 정렬</p>
<h2>4. 퀵 정렬 (재귀 함수를 이용) O(nlogn) ~ O(n^2)</h2>
<pre class="sql"><code>def q_sort(array, start, end):
  
    if end == start:
        return

    low = start
    high = end

    pivot = array[(low + high) // 2]
    while low &lt;= high:
        while array[low] &lt; pivot:
            low += 1
        while array[high] &gt; pivot:
            high -= 1
        if low &lt;= high:
            array[low], array[high] = array[high], array[low]
            low, high = low + 1, high - 1 

    mid = low
    q_sort(array, start, mid - 1)
    q_sort(array, mid, end)

def quick_sort(array):
    q_sort(array, 0, len(array)-1)
</code></pre>
<p><b>재귀 함수</b>를 사용해서 <b>pivot</b>값을 중심으로 반 씩 나눠 정렬하는 방식이다.</p>
<p>만약 정렬할 <b>배열의 길이가 1이하</b>인 경우 return해야한다. &rarr; <b>정렬할 요소가 없음.</b></p>
<p>정렬은 <b>low&le;high</b>일 경우 계속된다.</p>
<p>pivot값보다 <b>low값이 작은 경우</b> low+=1, <b>high값이 큰 경우</b> high-=1</p>
<p>low, high 비교를 끝내고 <b>low&le;high</b>일 경우 <b>low, high 스왑 후 low, high값 증감</b></p>

<h2>8. 선택 정렬의 개념</h2>
<p>여러 데이터 중에서 가장 작은 값을 뽑는 작동을 반복하여 값을 정렬하는 방식이다.</p>

<h2>11. 버블 정렬의 개념</h2>
<p>첫 번째 값부터 시작해서 바로 앞뒤 데이터를 비교하여 큰 것은 뒤로 보내는 방법을 사용한다.</p>
<p>각 사이클이 끝날 때마다 마지막 위치에 가장 큰 데이터가 자리잡는다.</p>
<p><b>데이터 1개를 제외하고 대부분 정렬되어 있는 배열일 경우</b> 연산 수가 급격히 줄어든다.</p>
<h2>12. 퀵 정렬의 개념</h2>
<p>**기준(pivot)**을 하나 뽑은 후 기준보다 <b>작은 그룹</b>과 <b>큰 그룹</b>을 나누어 다시 각 그룹을 정렬하는 방법이다.</p>
<p>나눈 그룹을 다시 정렬하고자 <b>재귀 호출</b>을 하고, 각 그룹의 정렬이 완료되면 합치는 방식을 사용한다.</p>
