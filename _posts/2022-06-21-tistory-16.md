<h2>1. 재귀호출 이용한 카운트다운 함수</h2>
<pre class="isbl"><code>def countDown(n):
    if n==0:
        print('발사!!')
    else:
        print(n)
        countDown(n-1)
</code></pre>
<h2>2. 재귀호출 이용한 별 출력 함수</h2>
<pre class="isbl"><code>def printStar(n):
    if n &gt; 0:
        printStar(n-1)
        print('&starf;' * n)
</code></pre>

<h2>4. n번째 피보나치 수 계산 함수</h2>
<pre class="angelscript"><code>def fibo(n):
    if n==0:
        return 0
    elif n ==1:
        return 1
    else:
        return fibo(n-1) + fibo(n-2)
</code></pre>
<h2>5. 주어진 10진수 숫자를 3/8/12/16진수로 변환하여 출력 함수</h2>
<pre class="python"><code>def notation(base, n, numberChar):
    """
    base: 진수
    n: 10진수 숫자
    numberChar: 12, 16진수 표현 문자
    """
    # 목표: 주어진 10진수 숫자를 3/8/12/16진수로 변환하여 출력
    if n &lt; base:
        print(numberChar[n], end = ' ')
    else:
        notation(base, n // base, numberChar)
        print(numberChar[n%base], end = ' ')
</code></pre>
