<h2>1. 배낭에 담을 수 있는 보석의 최대 가격 계산 함수</h2>
<pre class="vim"><code>def knapsack(maxWeight, rowCount, weight, money):
    """
    maxWeight: 배낭 최대 무게
    rowCount: 보석 개수
    weight: 보석 무게 리스트
    money: 보석 가격 리스트
    출력값: 배낭에 담을 수 있는 보석의 최대 가격
    """

    print('## 메모이제이션 배열 ##')

    # 코드 작성 목표: 메모이제이션 배열 생성, 배낭의 담을 수 있는 보석의 최대 가격 return
    array = [[0 for _ in range(maxWeight + 1)] for _ in range(rowCount + 1)]
    for row in range(1, rowCount + 1):
        print(f'{row} 개 --&gt; ', end=' ')
        for col in range(1, maxWeight + 1):
            if weight[row] &gt; col:
                array[row][col] = array[row - 1][col]
            else:
                value1 = money[row] + array[row - 1][col - weight[row]]
                value2 = array[row - 1][col]
                array[row][col] = max(value1, value2)
            print(f'{array[row][col]}', end=' ')
        print()
    return array[rowCount][maxWeight]
</code></pre>
<p>메모이제이션 배열을 (행)maxweight + 1와 (열)rowcount + 1 크기만큼 생성한다. &rarr; <b>+1 을 하는 이유는 0행, 0열에 0값을 저장하기 위해 지정한다.</b></p>
<p>row for문을 순회할 때는 0이 들어있는 열을 <b>제외</b>한 열부터 순회해야 하므로 <b>1, rowcount + 1</b>까지 순회한다. &rarr; <b>col</b>도 마찬가지이다.</p>
<p>col for문을 순회할 때는 0이 들어있는 열을 <b>제외</b>한 열부터 순회해야 하므로 <b>1, maxweight + 1</b>까지 순회한다. &rarr; <b>col</b>도 마찬가지이다.</p>
<p>만약 보석무게보다 가방 무게가 작을 때 바로 위의 데이터 값을 가져온다.</p>
<p>보석무게보다 가방 무게가 클 때 바로 위 값과 보석의 가격 + 위의 데이터 중 가방 무게 - 보석무게의 데이터를 비교해서 큰 값을 가져온다.</p>
<p>rowCount, maxWeight에 담겨있는 <b>최적의 값</b>을 반환한다.</p>
<h2>2. 황금 미로에서 얻은 최대 황금 개수 계산 함수</h2>
<pre class="inform7"><code>def growRich(ROW, COL, goldMaze):
    """
    ROW: 황금 미로 행 개수
    COL: 황금 미로 열 개수
    goldMaze: 황금 미로 좌표 별 황금 개수 행렬
    출력값: 황금 미로에서 얻은 최대 황금 개수
    """
    # 코드 작성 목표: 메모이제이션 배열 생성, 황금 미로에서 얻은 최대 황금 개수 return
    array = [[0 for _ in range(COL)] for _ in range(ROW)]
    array[0][0] = goldMaze[0][0]

    rowSum = array[0][0]
    for i in range(1, COL):
        rowSum += goldMaze[0][i]
        array[0][i] = rowSum
    
    colSum = array[0][0]
    for i in range(1, ROW):
        colSum += goldMaze[i][0]
        array[i][0] = colSum

    for i in range(1, ROW):
        for j in range(1, COL):
            if array[i][j - 1] &gt; array[i - 1][j]:
                array[i][j] = array[i][j - 1] + goldMaze[i][j]
            else:
                array[i][j] = array[i - 1][j] + goldMaze[i][j]
                
    return array[ROW - 1][COL - 1]
</code></pre>
<p>메모이제이션 배열 생성하고 초기 0행, 0열에 <b>열, 행누적값</b> 대입 &rarr; row부터</p>
<p><b>0행, 0열 제외</b>하고 for문을 순회하며 비교</p>
<p>메모이제이션의 <b>위 값</b>과 <b>왼쪽 값</b>을 비교해서 해당 <b>황금 미로의 인덱스값과 더해</b>서 <b>메모이제이션</b>의 해당 인덱스에 저장</p>
<p>마지막 인덱스가 <b>최적의 값</b>이므로 메모이제이션의 <b>ROW-1, COL-1</b>위치 값 반환</p>

<h2>4. 피보나치 수열 계산 함수</h2>
<pre class="angelscript"><code>def dp_fibo(n):
    # 동적계획법을 이용한 n번째 피보나치 수열 계산
    global count_dp
    memo = [1, 1]

    # 코드 작성 부분
    if n&lt;2:
        return memo[n]
    else:
        for i in range(2, n+1):
            memo.append(memo[i-1] + memo[i-2])
            count_dp += 1
        return memo[n]
</code></pre>
<p>초기 메모이제이션 배열 <b>[1, 1]</b></p>
<p>피보나치 수열의 1, 2번째는 1이여서 <b>n&lt;2 일 때 1반환</b></p>
<p><b>i : 2부터 n+1까지 순회</b> 메모이제이션 배열에 <b>i - 1, i - 2</b> 더해서 메모이제이션에 추가</p>
