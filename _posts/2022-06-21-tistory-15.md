<h2>2. 깊이 우선 탐색 함수</h2>
<pre class="gradle"><code>def find_vertex(g, find_vtx):
    stack = [] 
    visitedAry = [] 
    current = 0  
    stack.append(current)
    visitedAry.append(current)

    while len(stack) &gt; 0:
        next = None
        for vertex in range(g.SIZE):
            if g.graph[current][vertex] != 0:
                if vertex not in visitedAry:
                    next = vertex
                    break
        
        if next != None:
            current = next
            stack.append(current)
            visitedAry.append(current)
        
        else:
            current = stack.pop()

    if find_vtx in visitedAry:
        return True  
    else:
        return False  
</code></pre>

<h2>5. 그래프의 원리</h2>
<ul>
<li>그래프는 여러 노드가 연결된 자료구조이다.</li>
<li>간선의 방향성 여부에 따라 방향 그래프와 무방향 그래프로 나뉜다.</li>
<li>간선에 가중치를 부여해 가중치 그래프도 만들 수 있다.</li>
<li>트리의 노드에 해당하는 용어가 그래프에서는 정점이다.</li>
</ul>
