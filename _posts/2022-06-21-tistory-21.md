<h4><b>생선들의 데이터 분류 방법</b></h4>
<p>일단 생선의 특징을 알아야 합니다. = 길이, 무게등등</p>

<p>찾고자 하는 생선의 데이터(길이, 무게)를 리스트로 준비합니다.</p>

<p>머신러닝에서 여러개의 종류(클래스) 중 하나를 구별해 내는 문제를 <b>분류(classification)</b>이라고 부릅니다.</p>
<p>2개의 클래스중 하나를 고르는 문제는 <b>이진 분류(binary classification)</b>라고 합니다.</p>

<p>각 도미의 특징을 <b>길이와 무게</b>로 표현한 것을 <b>특성(feature)</b>라고 합니다.</p>

<p>두 특성을 <b>그래프</b>로 표현하면 데이터 이해가 쉽고 앞으로 할 작업에 대한 힌트를 얻을 수 있습니다.</p>

<p>파이썬에서 <b>과학계산용 그래프</b>를 그리는 대표적인 패키지는 <b>matplotlib</b>입니다.</p>
<p><b>plt.scatter()는 </b>x, y축을 나눠 점으로 <b>산점도</b>를 그릴 때 필요한 함수입니다.&nbsp;<b></b></p>
<p><b>plt.xlabel(), plt.ylabel()</b>은 x축, y축의 타이틀을 정해주는 함수입니다.</p>
<p><b>plt.show()</b> 그래프로 시각화하는 함수입니다.</p>

<p>아래 코드를 보면 이해가 쉽습니다.</p>

<p>위 그래프는 도미의 2개의 특성(길이, 무게)을 사용해 그린 그래프이기 때문에 <b>2차원 그래프</b>라고 합니다.</p>

<p>일반적으로 생선의 길이가 길수록 무게가 많이 나가므로 현재 그래프의 모습은 매우 자연스럽습니다.</p>

<p>이렇게 자연스러운, <b>일직선에 가까운 형태</b>로 나타나는 경우를 <b>선형적</b>이라고 말합니다.</p>

<p><b>빙어의 데이터</b>도 입력하고 그래프로 출력시</p>
<pre class="angelscript"><code>import matplotlib.pyplot as plt

bream_length = [25.4, 26.3, 26.5, 29.0, 29.0, 29.7, 29.7, 30.0, 30.0, 30.7, 31.0, 31.0,

<p>두 생선의 특성을 모두 합쳐 <b>2차원 리스트</b>를 만드는 코드</p>
<pre class="ini"><code>length = bream_length + smelt_length
weight = bream_weight + smelt_weight

fish_data = [[l, w]for l, w in zip(length, weight)]</code></pre>
<p>마지막으로 준비할 데이터는 <b>정답 데이터</b>입니다.</p>
<p>첫번째 생선은 도미이고 마지막 생선은 빙어인지 <b>1과 0으로</b> 알려주어야 합니다.</p>
<p>도미를 1, 빙어를 0으로 두겠습니다.</p>
<p>도미가 35마리, 빙어가 14마리이므로 <b>1이 35번, 0이 14번</b> 있는 데이터가 정답 데이터입니다.</p>

<p>이제 사이킷런 패키지에서 k-최근접 이웃 알고리즘을 구현한 클래스인 KNeighborsClassifier(<b>기본값은 5</b>)를 임포트합니다.</p>
<p>KNeighborsClassifier 객체를 생성합니다.</p>
<p>이 객체에 <b>fish_data와 fish_target</b>를 전달하여 <b>도미를 찾기 위한 기준</b>을 학습시킵니다.</p>

<p>이러한 과정을 머신러닝에서는 <b>훈련(training)</b>이라고 부릅니다. = 사이킷런에서는 <b>fit()</b>함수가 이런 역할을 합니다.</p>
<pre class="css"><code>kn.fit(fish_data, fish_target)</code></pre>
<p>머신러닝 알고리즘을 구현한 프로그램을 <b>모델(model)</b>이라고 부릅니다.</p>
<p>프로그램이 아니더라도 <b>알고리즘을 구체화하여 표현한것</b>도 <b>모델</b>이라고 부릅니다</p>

<p>사이킷런에서 모델을 평가하는 함수는 <b>score()</b>입니다.</p>
<p><b>0~1 사이의 값</b>을 반환하고 다 맞출 경우 1을 반환합니다.</p>
<p>반환된 값을 <b>정확도</b>라고 합니다. </p>
<p>정확도 = (맞힌 개수) / (전체 데이터 개수)</p>

<p><b>k-최근접 이웃 알고리즘</b>은 데이터에 대한 답을 구할 때 <b>주위의 다른 데이터</b>를 보고 <b>다수의 데이터가 해당</b>하는 값을 정답으로 사용합니다.</p>
<p>데이터가 크기 때문에 <b>메모리가 많이 사용</b>되고 직선거리를 계산하는데 <b>많은 시간이 소요</b>됩니다.</p>
<p>이러한 특성 때문에 <b>데이터가 많을 경우 해당 알고리즘을 사용하기 어렵습니다.</b></p>

<p>KNeighborsClassifier(<b>기본값은 5</b>)이 기본값을 KNeighborsClassifier(<b>n_neighbor=n</b>) 을 사용해서 변경 가능합니다.</p>
