<h3>점진적 학습</h3>
<p>이전에 학습한 모델을 <b>버리지</b> 않고 새로운 데이터셋에 대해서 조금씩 훈련</p>
<p>대표적인 <b>점진적 학습 알고리즘</b>은 <b>확률적 경사 하강법</b>입니다.</p>

<h3>확률적 경사 하강법</h3>
<p>경사를 따라 내려가는 방법</p>
<p>가파른 경사를 따라 <b>원하는 지점</b>에 도달하는것이 목표</p>
<p>가파른 길을 찾아 내려오지만 조금씩 내려오는게 중요 &rarr; <b>경사 하강법 모델</b>을 훈련</p>
<p>전체 샘플을 사용하지 않고 <b>하나의 샘플만</b>을 가지고 가파른 길을 찾는 법 &rarr; <b>확률적 경사 하강법</b></p>
<p>훈련 세트를 <b>한번 모두 사용</b>하는 과정을 <b>에포크</b>, 일반적으로 수십, 수백번 수행</p>

<ol>
<li><b>1개의 샘플</b>을 사용해 경사 하강법을 수행하는 방식이 <b>확률적 경사 하강법</b></li>
</ol>

<ol>
<li><b>여러 개의 샘플</b>을 사용해 경사 하강법을 수행하는 방식이 <b>미니배치 경사 하강법</b></li>
<li>&nbsp;<b>실전</b>에서 아주 많이 사용하는 방법이다.</li>
</ol>

<ol>
<li><b>전체 샘플</b>을 사용해 경사 하강법을 수행하는 방식은 <b>배치 경사 하강법</b></li>
<li>전체 샘플을 사용해 <b>가장 안정적인 방법</b>이고 <b>컴퓨터 자원을 많이 사용</b>하는 방법.</li>
</ol>

<p><figure class="imageblock alignLeft"><img alt="" loading="lazy" src="/assets/img/tistory/23/image-01.png"/></figure>
</p>
<p>훈련 데이터가 준비되어 있지 않고 <b>매일매일 업데이트 되어도</b> 학습을 계속 <b>이어나갈 수</b> 있습니다.</p>
<h3>손실 함수</h3>
<p>문제에서 머신러닝 <b>알고리즘</b>이 얼마나 잘못되었는지 <b>측정</b>하는 <b>기준</b></p>
<p>손실 함수의 값이 작을수록 좋지만, 어떤값이 최소값인지는 알 수 없다.</p>

<p><b>분류</b>에서의 손실함수, 정답을 못 맞추는 것</p>
<p>정확도는 <b>예측중에 맞은 개수 / 전체 샘플의 개수</b></p>

<p>샘플이 4개만 있다고 가정할시 가능한 정확도는 0(맞은 개수: 0), 0.25(맞은 개수: 1), 0.5(맞은 개수: 2) 0.75(맞은 개수: 3), 1(맞은 개수: 4) 이 전부이다.</p>
<p>이러한 정확도들은 듬성듬성하므로 <b>연속적이지 않다.</b></p>

<h3>로지스틱 손실 함수</h3>
<p>&lsquo;<b>이진 분류에서의 로지스틱 손실 함수&rsquo;</b></p>
<p><b>양성 클래스</b></p>

<p>예측&nbsp; &nbsp; &nbsp; 정답&nbsp; &nbsp; &nbsp; 손실함수</p>
<p>0.9&nbsp; &nbsp; &nbsp;&nbsp; &nbsp; 1&nbsp; &nbsp; &nbsp;&nbsp; &nbsp; &nbsp; -0.9</p>

<p><b>예측값</b>과 <b>타깃값</b>을 곱한 후 <b>음수로</b> 바꿔주면 <b>손실값(손실함수)</b>이 나온다.</p>
<p>타깃이 <b>양성 클래스</b>이므로 예측이 <b>1에 근접</b>해야 좋은 모델 &rarr; 손실값도 -0.9이므로 <b>작은 손실 값</b></p>

<p><b>음성클래스</b></p>

<p>예측&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 정답&nbsp; &nbsp; &nbsp;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;손실함수</p>
<p>0.2&nbsp; &nbsp; &nbsp;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 0&nbsp; &nbsp; &nbsp;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; ?</p>
<p>1에서 빼준다.&nbsp; &nbsp; &nbsp;1로 변환한다.&nbsp; &nbsp; -0.8</p>
<p>0.8&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;1&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;-0.2&nbsp;&nbsp;</p>

<p><b>음성 클래스</b>일 경우 <b>예측값을 1에서 빼주고 타깃값을 1로 변환한 후</b> 양성 클래스와 같게 계산한다.</p>
<p>타깃이 <b>음성 클래스</b>이므로 예측이 <b>0에 근접</b>해야 좋은 모델 &rarr; 손실값도 -0.8이므로 <b>작은 손실 값</b></p>

<p>예측 확률에 <b>로그함수</b>를 적용하면 <b>더 보기가 편하다</b>.</p>

<p>예측범위는 0~1 사이인데, <b>로그함수</b>는 이 사이에서 음수가 되므로 <b>최종 손실값</b>은 <b>양수</b>가 된다.</p>
<p>로그함수는 0에 가까울수록 아주 큰 음수가 되기 때문에 손실을 크게 만들어 모델에 영향이 커진다.</p>
<p><figure class="imageblock alignLeft"><img alt="" loading="lazy" src="/assets/img/tistory/23/image-02.png"/></figure>
<figure class="imageblock alignLeft"><img alt="" loading="lazy" src="/assets/img/tistory/23/image-03.png"/></figure>
</p>
<p>&rarr; 이러한 함수를 <b>로지스틱 손실 함수</b>라고 부른다. 또는 <b>이진 크로스엔트로피 손실 함수</b>라고 부른다.</p>

<p>&lsquo;<b>다중 분류</b>에서의 <b>로지스틱 손실 함수&rsquo;</b></p>
<p>다중 분류에서의 손실함수는 <b>크로스엔트로피 손실 함수</b>를 사용한다.</p>

<p><b>회귀</b>에서의 손실 함수는 <b>평균 절댓값 오차, 평균 제곱 오차</b>를 많이 사용한다.</p>
<p><b>평균 절댓값 오차</b>: 타깃에서 예측을 뺀 <b>절댓값</b>을 <b>모든 샘플에 평균</b></p>
<p><b>평균 제곱 오차</b>: 타깃에서 예측을 뺀 값을 <b>제곱한</b> 후 <b>모든 샘플에 평균</b></p>

<h3>확률적 경사 하강법을 사용한 분류 모델 생성</h3>
<ol>
<li><b>csv파일</b>로 <b>데이터프레임</b>을 만들어준다.</li>
<li><b>데이터프레임</b>에서 필요한 <b>열</b>들을 <b>입력, 타깃 데이터</b>로 나눠 <b>넘파이 배열</b>로 만들어준다.</li>
<li>사이킷런의 <b>train_test_split()</b> 함수로 데이터를 <b>훈련세트와 테스트 세트</b>로 나눈다.</li>
<li>훈련세트와 테스트 세트를 표준화 전처리한다.
<ol>
<li><b>훈련세트로 학습한 통계 값</b>으로 <b>훈련세트와 테스트 세트를 변환</b>해야 한다.</li>
</ol>
</li>
</ol>
<pre class="makefile"><code>from sklearn.preprocessing import StandardScaler

ss = StandardScaler()
ss.fit(train_input)
train_scaled = ss.transform(train_input)
test_scaled = ss.transform(test_input)</code></pre>
<p><b>확률적 경사 하강법</b> 제공하는 <b>분류용 클래스(SGDClassifier)</b> 생성</p>
<pre class="python"><code>from sklearn.linear_model import SGDClassifier

#loss는 손실함수의 종류 지정, log로 로지스틱 손실 함수 지정
#max_iter에서 수행할 에포크의 횟수 지정
sc = SGDClassifier(loss='log', max_iter=10, random_state=42)
sc.fit(train_scaled, test_scaled)</code></pre>
<p><b>확률적 경사 하강법</b>은 <b>점진적인 학습</b>이 가능하다.</p>
<p>추가로 모델을 훈련하고 싶을 경우 <b>객체.partial_fit() </b>함수로 <b>이어서</b> 훈련한다.</p>
<p>호출시 <b>1에포크(모든 샘플 1번 로테이션)씩</b> 이어서 훈련한다.</p>


<h3><b>에포크와 과대/과소적합</b></h3>
<p><b>과대적합</b>: 훈련세트의 정확도 <b>&gt;</b> 테스트세트의 정확도, 훈련이 훈련세트에만 너무 잘됨.</p>
<ol>
<li><b>많은 에포크 횟수</b>&nbsp;동안 훈련한 모델은 훈련세트에 너무 잘 맞아&nbsp;<b>과대적합 모델</b><b></b></li>
</ol>
<p><b>과소적합</b>: 훈련세트의 정확도 <b>&le;</b> 테스트 세트의 정확도, 둘다 매우 낮은 경우, 훈련 덜 됨.</p>
<ol>
<li><b>적은 에포크 횟수</b>&nbsp;훈련한 모델은&nbsp;<b>과소적합 모델</b><b></b></li>
</ol>
<p><figure class="imageblock alignLeft"><img alt="" loading="lazy" src="/assets/img/tistory/23/image-04.png"/></figure>
</p>
<p><b>훈련세트</b>는 에포크가 진행될수록 <b>꾸준히 증가</b></p>
<p>하지만 <b>테스트세트</b> 점수는 <b>최적점</b>을 지나고부터 <b>감소하기 시작</b></p>
<p><b>과대적합이 시작되기 전</b>에 훈련을 <b>멈추는</b> 것을 <b>조기종료</b> 라고 한다.</p>


<p><b>코드로 구현시</b></p>
<ol>
<li>300번의 에포크동안 훈련을 반복하여 <b>훈련결과를 리스트에 저장한다.</b></li>
<li>저장한 리스트를 **plt.plot()**으로 그려본다.</li>
<li><b>최적점</b>이 <b>100번째 에포크</b>라고 가정하자.</li>
</ol>

<p><figure class="imageblock alignLeft"><img alt="" loading="lazy" src="/assets/img/tistory/23/image-05.png"/></figure>
</p>
<p>&nbsp; &nbsp; &nbsp; 4. 위 그래프에서 <b>최적점을 지나면서</b> 훈련세트와 테스트 세트의 <b>점수가 조금씩 벌어지는 것</b>을 볼 수 있다.</p>
<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;에포크 초기에는 <b>과소적합</b>되어 훈련세트와 테스트 세트의 점수가 <b>모두 낮다.</b></p>
<p>&nbsp; &nbsp; &nbsp; 5. 이러한 경우 <b>100번째 에포크가 최적점</b>(적절한 반복 횟수)라고 말할 수 있다.</p>

<p><b>SGDClassifier</b>의 <b>매개변수인 tol</b>을 <b>None</b>으로 지정시 <b>멈추지 않고 학습</b>하는것을 의미한다.</p>

<p><b>SGDClassifier</b>의 <b>매개변수인 loss</b> 매개변수의 기본값은 &lsquo;hinge&rsquo; 힌지손실이다.</p>
<p><b>힌지손실</b>은 <b>서포트 벡터 머신</b> 이라는 다른 머신러닝 알고리즘을 위한 손실함수이다.</p>
