<h1><b>로지스틱 회귀</b></h1>
<h3>데이터 준비하기</h3>
<ol>
<li>CSV 파일을 읽을 경우 <b>데이터프레임</b> 형태로 저장된다.</li>
<li>데이터프레임에서 사용할 열을 <b>입력데이터</b>와 <b>타깃데이터</b>로 구분해 <b>넘파이 배열</b>로 변환한다.</li>
<li>분류한 입력과 타깃을 사이킷런의 <b>train_test_split</b> 함수를 활용하여 <b>훈련데이터</b>와 <b>테스트데이터</b>로 분할한다.
<ol>
<li>from sklearn.model_selection import train_test_split</li>
</ol>
</li>
<li>사이킷런의 <b>standardscaler</b>를 사용하여 훈련데이터와 테스트데이터를 <b>전처리(표준화)</b>해준다.
<ol>
<li>standardscaler 객체 생성후 <b>fit 함수</b>로 훈련 후 <b>transform 함수</b>로 변환</li>
<li><b>transform 함수로 테스트세트와 훈련세트 모두 변환해줘야 한다.</b></li>
</ol>
</li>
</ol>
<p>타깃데이터에 2개 이상의 클래스이 포함된 문제를 <b>다중 분류</b>라고 한다.</p>
<p>타깃값은 <b>객체명.classes_</b> 속성에 저장되어 있다.</p>
<p>사이킷런의 분류 모델은 <b>predict_proba()로 클래스별 확률값</b>을 반환한다.</p>
<p><b>&nbsp; &nbsp; &nbsp; classes_</b> 속성마다의 확률값들을 반환한다.</p>

<p><b>k-최근접 이웃 분류 알고리즘</b>은 <b>설명하기가 복잡하다.</b></p>

<h2>로지스틱 회귀</h2>
<p><b>선형 방정식</b>을 학습하는 <b>분류</b> 모델이다.</p>
<p>z = a x (특성1) + b x (특성2) + c x (특성3) + d x (특성4) + e x (특성5) + f</p>
<p>빨간색은 <b>가중치(특성 1~5)</b>이고 <b>파란색(f)</b>은 절편이다.</p>
<p><b>시그모이드 함수(로지스틱 함수)</b></p>
<p><b>&nbsp; &nbsp; &nbsp; z</b>가 <b>아주 큰 음수</b>일경우 <b>0</b>이 되고 <b>아주 큰 양수</b>일 경우 <b>1</b>로 설정한다.</p>
<p><figure class="imageblock alignLeft"><img alt="" loading="lazy" src="/assets/img/tistory/22/image-01.png"/></figure>
</p>
<p>z가 <b>무한한 음수</b>일 경우 함수는 <b>0에 근접</b>하고, <b>무한한 양수</b>일 경우 <b>1에 근접</b>한다는 특성이 있다.</p>
<p>z가 <b>0.5</b>보다 <b>작거나 같은</b> 경우 <b>음성 클래스</b>, 반대의 경우 <b>양성 클래스</b>로 판별한다.</p>

<h2>로지스틱 회귀로 이진 분류 수행</h2>
<p><b>불리언 인덱싱</b>: 넘파이 배열의 <b>key 값대신 True, False값</b>으로 <b>value 값</b>들을 불러올 수 있다.</p>
<p>&nbsp; &nbsp; &nbsp; ex: arr[[True, False, True, True, True]]</p>

<p><b>로지스틱 회귀</b> 모델 훈련: <b>LogisticRegression</b>객체 생성 후 <b>fit 함수</b>로 훈련 후 <b>predict 함수</b>로 예측</p>
<p>&nbsp; &nbsp; &nbsp; from sklearn.linear_model import LogisticRegression</p>
<p>로지스틱 회귀가 학습한 계수 확인하기 위해 <b>가중치(coef_), 절편(intercept_)</b> 확인</p>

<p><b>로지스틱 회귀로 계산한 샘플의 z값</b> 출력시 <b>객체.decision_function</b> <b>함수</b> 사용</p>

<p><b>decision_function</b>에서 나온 값을 <b>로지스틱 함수</b>에 적용시키면 <b>확률</b>을 얻을 수 있다.</p>
<p><b>scipy 라이브러리</b>에 시그모이드 함수가 존재한다.</p>
<p>&nbsp; &nbsp; &nbsp;&nbsp;from scipy.special import expit</p>
<p>&nbsp; &nbsp; &nbsp;&nbsp;expit([샘플들의 z값 리스트])</p>

<h2>로지스틱 회귀로 다중 분류 수행</h2>
<p><b>LogisticRegression</b> 클래스의 매개변수인 <b>max_iter 은 모델 훈련 반복횟수</b>를 지정한다.</p>
<p><b>디폴트값은 100이다.</b> 모델을 충분히 학습 시키고 싶을 경우 수를 늘려주면 된다.</p>

<p><b>LogisticRegression</b>는 기본적으로 <b>릿지 회귀</b>와 같이<b> 계수의 제곱(L2)</b>을 규제한다.</p>
<p><b>&nbsp; &nbsp; &nbsp;&nbsp;규제를 제어하는 매개변수</b>는 <b>C</b> 이며 <b>디폴트 값은 1이다.</b></p>
<p><b>&nbsp; &nbsp; &nbsp;&nbsp;값이 작아질수록</b> <b>규제가 강력</b>해지고 <b>값이 커질경우 규제가 약해진다.</b></p>
<p>&nbsp; &nbsp; &nbsp;&nbsp;다른 규제 매개변수인 <b>alpha</b>와는 <b>반대로</b> 작동한다.</p>

<p>다중분류에서 <b>predict_proba()</b>의 결과중에 <b>가장 높은 확률</b>이 예측 클래스가 된다.</p>

<p>다중분류는 <b>클래스마다 z값</b>을 하나씩 계산한다.</p>
<p><b>가장 높은 z값</b>을 출력하는 클래스가 <b>예측 클래스</b>가 된다.</p>

<p><b>다중분류</b>에서의 <b>확률 계산</b>시 <b>소프트맥스 함수</b>를 사용해 n개의 z값을 확률로 변환한다.</p>
<p><b>&nbsp; &nbsp; &nbsp;&nbsp;여러개의 선형 방정식</b>의 출력값을 <b>0~1사이의 값으로 압축</b>하고 <b>전체 합이 1</b>이 되도록 만듬.</p>
<p>&nbsp; &nbsp; &nbsp;&nbsp;이 과정을 위해서 지수함수를 사용해 <b>정규화된 지수 함수</b>라고도 부릅니다.</p>
<p><figure class="imageblock alignLeft"><img alt="" loading="lazy" src="/assets/img/tistory/22/image-02.png"/></figure>
</p>
<p>사이파이에서 <b>소프트맥스 함수</b> 사용</p>
<p>&nbsp; &nbsp; &nbsp;&nbsp;from scipy.special import softmax</p>
<p>&nbsp; &nbsp; &nbsp;&nbsp;a = softmax([샘플들의 z값 리스트], axis = 1)</p>
<p><b>&nbsp; &nbsp; &nbsp;&nbsp;&nbsp; &nbsp; &nbsp;&nbsp;axis 매개변수</b>는 각 <b>행</b> 샘플에 관해서 소프트맥스 함수 계산<br/><br/></p>
