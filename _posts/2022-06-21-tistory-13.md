<h2>1. 큐가 꽉 차있는지 확인하는 함수&nbsp;</h2>
<pre class="monkey"><code>def is_queue_full(self):
        if self.size - 1 != self.rear:
            return False

        elif self.size -1 == self.rear and self.front == -1:
            return True
        else:
            for i in range(self.front + 1, self.size):
                self.queue[i - 1] = self.queue[i]
                self.queue[i] = None
            self.front -= 1
            self.rear -= 1
            return False
</code></pre>
<h2>2. 큐가 비었는지 확인하는 함수</h2>
<pre class="ruby"><code>def is_queue_empty(self):
        if self.front == self.rear:
            return True
        else:
            return False
</code></pre>
<h2>3. 큐에 데이터를 삽입하는 enQueue 함수</h2>
<pre class="ruby"><code>def en_queue(self, data):
        if (self.is_queue_full()):
            return 
        self.rear += 1
        self.queue[self.rear] = data
</code></pre>
<h2>4. 큐에서 데이터를 추출하는 deQueue 함수</h2>
<pre class="ruby"><code>def de_queue(self):
        if (self.is_queue_empty()):
            return None
        self.front += 1
        data = self.queue[self.front]
        self.queue[self.front] = None
        return data
</code></pre>
<h2>5. 큐에서 front값을 확인하는 peek 함수</h2>
<pre class="ruby"><code>def peek(self):
        if (self.is_queue_empty()):
            return None
        return self.queue[self.front + 1]
</code></pre>
<h2>6. 실습 과제 2번 - 시간 계산 함수</h2>
<pre class="ruby"><code>def calc_time(self):
        sum = 0
        for i in range(self.front + 1, self.rear + 1):
            sum += self.queue[i][1]
        return sum
</code></pre>
<h2>7. 큐의 원리</h2>
<ul>
<li>큐는 <b>양쪽이</b> 뚫려 있는 구조이다.</li>
<li>큐는 <b>선입선출(FIFO)</b> 구조이다.</li>
<li>한쪽에서는 <b>삽입만</b>, 다른 쪽에서는 <b>추출만</b> 진행된다.</li>
<li>데이터를 <b>삽입</b>할때는 <b>rear + 1</b> 위치에 삽입된다.</li>
<li>데이터를 <b>추출</b>할때는 <b>front + 1</b>위치가 추출된다.</li>
</ul>
